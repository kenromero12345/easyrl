EasyRL Web Application

Software Requirements:

Our code can be run on Google Chrome.

Requires Python 3.7 and pytorch 1.6

Our code follows the MVC design.

Python is used as the model. HTML is used as the view. Flask is used as the controller.

To help the view part of the design, CSS, Django, JavaScript, JQuery, and Ajax are used.

Further requires Tensorflow 2.1, Keras, Kivy and other packages, see Readme.txt for an explanation and requirements.txt for details.
